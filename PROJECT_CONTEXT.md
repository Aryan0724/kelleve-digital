FindMyInterior

# PROJECT DESCRIPTION

FindMyInterior is Bihar's Home Improvement, Interior, Construction, Supplier, Builder and Workforce Marketplace.

The platform connects:

* Homeowners
* Interior Designers
* Architects
* Builders
* Contractors
* Suppliers
* Skilled Workers

The platform should feel like:

Houzz + Urban Company + IndiaMART + Architizer + Urban Ladder

but focused entirely on Bihar.

---

# PRIMARY GOAL

Generate Leads.

The platform is NOT a social network.

The platform is NOT a project management software.

The platform is NOT a CRM.

The platform is a marketplace and lead-generation engine.
