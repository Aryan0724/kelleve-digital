# FIND MY INTERIOR - VERIFICATION & PROFILE COMPLETION SPECIFICATION

## Purpose

This document defines how every user completes their profile, becomes verified, and builds trust on the platform.

Verification is completely **manual** in Phase 1.

There is NO automatic verification.

Only an Admin can approve or reject verification requests.

The objective is to make every public profile trustworthy and ensure homeowners and businesses can confidently hire professionals.

---

# GENERAL PRINCIPLES

Every role has different verification requirements.

Different users perform different work.

Therefore different users must provide different information.

No role should see unnecessary fields.

The profile completion system must guide users step-by-step until they reach 100%.

Users should always know:

- What information is missing
- What documents are pending
- Why verification was rejected
- What benefits verification provides
