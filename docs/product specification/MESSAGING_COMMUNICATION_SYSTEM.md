# FIND MY INTERIOR - MESSAGING & COMMUNICATION SYSTEM

## Purpose

Messaging is not a standalone chat application.

Every conversation on Find My Interior must be attached to a Project, RFQ, Job or Opportunity.

Users should never randomly message anyone.

Every conversation must have a business purpose.

The messaging system should help users complete projects, not simply chat.

---

# GENERAL PRINCIPLES

Every conversation belongs to

• Project

• RFQ

• Job

• Order

• Builder Project

There should never be random inbox conversations.

Every chat must be connected to an opportunity.
