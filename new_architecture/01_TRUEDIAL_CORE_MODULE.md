# TRUEDIAL CORE MODULE
Version 1.0

---

# PURPOSE

TRUEDIAL is Tenant #2 on the existing platform.

Find My Interior already exists.

The objective is NOT to create another Laravel project.

The objective is to extend the existing platform.

Everything possible must reuse existing platform services.

---

# REUSE EXISTING PLATFORM

DO NOT recreate:

Authentication

Users

Roles

Permissions

Media Upload

Reviews

Notifications

Payments

Wallet

Settings

Activity Logs

Storage

Search Infrastructure

API Authentication

Admin Authentication

These already exist inside the platform.

TRUEDIAL must consume them.
